# ⚛ Mekanism Fusion Reactor — Fuel Calculator

Calculate D-T Fuel consumption and energy output of the **Mekanism** (Minecraft) fusion reactor based on the injection rate.

> **Vibe coded** with Claude Code — formulas sourced from the [FTB Wiki](https://ftb.fandom.com/wiki/Fusion_Reactor_(Mekanism)), UI generated by AI.

**Live:** [lauwed.github.io/mekanism-fission-reactor-calculator](https://lauwed.github.io/mekanism-fission-reactor-calculator/)

**Links:** [FTB Wiki](https://ftb.fandom.com/wiki/Fusion_Reactor_(Mekanism)) · [Mekanism on CurseForge](https://www.curseforge.com/minecraft/mc-mods/mekanism) · [GitHub](https://github.com/Lauwed/mekanism-fission-reactor-calculator)

## Features

- **Injection rate** slider (2–98, even numbers only) for internal D + T mixing
- **D-T Fuel flow rate** slider (1–1,000 mB/t) for direct pre-mixed fuel injection
- Cooling method toggle: **air-cooled** or **water-cooled** (+ Industrial Turbine)
- Deuterium, Tritium and D-T Fuel consumption per tick / second / minute / hour
- Energy output in FE/t with turbine + direct breakdown for water-cooled
- Detailed formula cards covering all reactor mechanics

## Stack

- [Vite](https://vite.dev/) + [React 19](https://react.dev/)
- Vanilla CSS, zero UI dependencies
